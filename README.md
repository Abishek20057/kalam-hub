# KalamHub

AI-powered innovation ecosystem — Home Page Module 1 foundation.

## Stack

- Next.js 15 (App Router) + React 19
- TypeScript (strict)
- Tailwind CSS v4
- shadcn/ui (New York style)
- Framer Motion
- Lucide React
- Prisma ORM (PostgreSQL) — schema defined, not yet migrated (Phase 5)

## Getting Started

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
